\# Multi-Disease Detection using Deep Learning



This project detects:

\- Pneumonia (Chest X-ray)

\- Tuberculosis (Chest X-ray)

\- Diabetic Retinopathy (Fundus images)



\## Models Used

\- ResNet50

\- DenseNet

\- EfficientNet

\- ViT + BiLSTM

\- Grad-CAM for explainability



\## Tech Stack

\- Python

\- PyTorch

\- TensorFlow / Keras

\- Flask

\- OpenCV



\## How to Run

1\. Install requirements

2\. Run app.py

3\. Upload image via web UI
